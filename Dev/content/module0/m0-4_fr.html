Description du cours

Le cours est composé de trois modules et d’un test.

  • Module 1 : Le « quoi » et le « pourquoi » de la gestion de l’information
  • Module 2 : Les ressources documentaires
  • Module 3 : La gestion des messages électroniques
  • Test final

Pour réussir ce cours, vous devez obtenir une note d’au moins 80 % au test. Si vous n’obtenez pas la note de passage, ne vous inquiétez pas. Vous pouvez refaire le test autant de fois que nécessaire.

Il vous faudra environ 60 minutes pour suivre le cours.

Bon succès!
