Les médias sociaux et l’information

Les médias sociaux occupent déjà une grande place dans notre vie de tous les jours. Avec le temps, ils trouvent également une place dans notre vie professionnelle.

À quelle fréquence consultez-vous les plateformes sociales du gouvernement du Canada comme GCconnex ou GCpedia? Utilisez-vous ces sites pour partager de l’information, pour collaborer sur un projet ou pour faire du réseautage?

Lorsque vous le faites, n’oubliez pas que si vous prenez une décision ou une mesure dans le cadre d’une conversation électronique sur les médias sociaux, vous devez consigner la décision ou la mesure pour vous assurer que l’information est stockée dans le dépôt ministériel de votre organisation.

La plus grande partie de l’information affichée sur les plateformes des médias sociaux a toutefois probablement déjà été stockée dans de la documentation officielle, comme les sites Web, les comptes rendus de réunions ou les plans de projets. L’information utilisée dans les médias sociaux est donc éphémère quand elle est déjà saisie dans d’autres documents officiels et il faut la traiter en conséquence.
