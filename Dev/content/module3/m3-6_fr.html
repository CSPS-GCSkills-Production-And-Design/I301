Résumé du module

Dans ce module, vous avez appris pourquoi et comment vous devez gérer vos messages électroniques. Voici les principaux points présentés :

  • Il vous incombe de gérer tous vos messages électroniques, comme vous le faites pour toute autre ressource d’information.
  • Vous devez traiter les messages comme la propriété du gouvernement et utiliser un bloc-signature normalisé et bilingue dans tous les courriels que vous envoyez.
  • Vous êtes responsable de stocker les messages électroniques ayant une valeur opérationnelle dans le dépôt ministériel désigné.
  • Vous devez supprimer régulièrement tout message électronique éphémère dont vous n’avez plus besoin.
  • Lorsque vous utilisez les médias sociaux pour le travail ou pour des objectifs personnels, vous devez suivre les règles et les lignes directrices établies par votre organisation.
Vous pouvez maintenant passer au test final.
