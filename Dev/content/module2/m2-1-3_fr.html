Types de ressources documentaires

Les types de ressources documentaires dépendent de leur fonction. À titre d’employé du gouvernement du Canada, il peut vous arriver notamment de rédiger des rapports, d’organiser des réunions ou de communiquer des décisions.

Voici des exemples de types de ressources documentaires :

  • les notes de service
  • les rapports
  • les comptes rendus de réunions
  • les factures
  • les contrats
  • les bases de données
  • les wikis

Cette liste n’est pas complète. N’oubliez pas que selon la définition, les ressources documentaires comprennent « toute production documentaire sous forme publiée ou non, quels que soient … le mode de production ou le support d’enregistrement. »
