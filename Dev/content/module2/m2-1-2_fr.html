Sources de l’information

Dans le cadre de votre travail quotidien, vous pouvez recevoir, analyser ou chercher de l’information qui provient de différentes sources.

Voici quelques-unes des sources ou origines des ressources documentaires :

  • les personnes
  • les organisations
  • les documents
  • les images
  • les sites Web
  • les enregistrements
  • les observations
  • les expériences scientifiques

La liste n’est pas complète. N’oubliez pas que selon la définition, les ressources documentaires comprennent « toute production documentaire … quelle que soit la source de communication … »
